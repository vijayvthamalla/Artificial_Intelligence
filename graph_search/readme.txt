Python is used to code for Part 1 and since it is in v3, it is not compatible with omega

results function will print the results of search algorithm
path function will print route along with cost
graph_search function will perform the search. It will perform A-star search for informed search and Uniform Cost Search for uninformed search 

To run the program:

    uninformed search:

        python main.py input1.txt Bremen Kassel
  

    informed search:

        python main.py input1.txt Bremen Kassel h_kassel.txt
